# Wave gate evidence packs

`/gate-check W<N>` writes `W<N>-evidence.md` here — the single markdown pack the founder
judges at each wave exit gate (rule 4). The founder reads the pack and answers "do these
numbers match reality? would I trust this score?" — never reads raw diffs.

Each pack contains: the wave's exit-gate checklist with pass/fail, test + E2E output, the
tenant-isolation proof, the golden dogfooding comparison, dashboard screenshots, instrumented
timings (e.g. W2's <10-min funnel), and adversarial-review findings with dispositions.
