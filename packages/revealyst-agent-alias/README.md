# revealyst-agent (alias)

This package is a thin alias for the canonical
[`@revealyst/agent`](https://www.npmjs.com/package/@revealyst/agent) so that
`npx revealyst-agent …` keeps working, and so the unscoped name cannot be
claimed by anyone else. It contains no logic of its own — the bin delegates
to `@revealyst/agent`.

See the canonical package for documentation.
